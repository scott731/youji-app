# 友记 Youji

根据《友记》产品需求文档实现的**朋友信息管理** Web 应用，核心理念：「让每一份友谊都被用心记住」。

## 功能概览

- **首页**：问候语、近期事件提醒、最近更新的朋友动态
- **朋友管理**：按分组筛选、拼音首字母排序、搜索；添加/编辑/删除朋友，完整基础信息与职业、个人特征、备注标签
- **朋友详情**（四个 Tab）  
  - **概况**：基本信息、职业信息、标签、备注  
  - **家庭**：家庭成员、子女教育信息（学校、类型、年级等），支持添加成员与教育记录  
  - **事件**：重要事件时间线（生日、婚嫁、升学、换工作等），支持添加事件与提醒  
  - **礼金**：礼金往来记录、往来差额、添加收支记录
- **事件日历**：月历视图、按日期展示事件、即将到来事件列表
- **我的**：修改昵称（首页问候）、数据概览、导出备份（JSON）

数据全部保存在**浏览器本地**（localStorage），无需后端，适合个人使用与备份。

## 技术栈

- React 18 + TypeScript
- Vite
- React Router
- Tailwind CSS

## 运行方式

确保已安装 [Node.js](https://nodejs.org/)（建议 18+），在项目目录执行：

```bash
cd youji-app
npm install
npm run dev
```

浏览器访问终端提示的地址（一般为 `http://localhost:5173`）。

打包：

```bash
npm run build
```

生成文件在 `dist/`，可部署到任意静态托管。

## 设计说明

- 主色：温暖橙 `#FF9500`，辅色：柔和蓝 `#5AC8FA`，背景米白、卡片白，与 PRD 视觉规范一致
- 底部导航：首页、朋友、添加、日历、我的；详情/编辑页隐藏底部栏
- 交互：列表可点击进入详情，详情内 Tab 切换概况/家庭/事件/礼金，表单弹窗添加家庭成员、事件、礼金
